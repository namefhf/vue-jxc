

## Project setup
```
npm install // or yarn
```

### Compiles and hot-reloads for development
```
npm run dev // or yarn dev

# 指定端口
npm run dev --port=8080 // or yarn dev --port=8080
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
